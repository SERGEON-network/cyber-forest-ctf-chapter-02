Upload CTF to GitHub

A complete guide to uploading your Cyber Forest CTF project to GitHub

1 Create a GitHub Account

If you don't already have a GitHub account, you'll need to create one to upload your project.

Pro Tip: Choose a username that represents you professionally as you may use GitHub for future projects too.

1. Go to github.com

2. Click "Sign up" in the top right corner

3. Enter your details and create your account

4. Verify your email address

2 Create a New Repository

A repository is where your project files will be stored on GitHub.

1. Click the "+" icon in the top right and select "New repository"

2. Name your repository (e.g., "cyber-forest-ctf")

3. Add a description: "A cybersecurity capture the flag challenge"

4. Choose Public (so others can see it)

5. Initialize with a README (important!)

6. Add .gitignore: Select "Node" if you have JavaScript files

7. Choose a license if you want

8. Click "Create repository"

3 Prepare Your Files

Organize your CTF project files before uploading.

Your CTF Project Structure

  • index.html Main file
  • style.css Styles
  • script.js JavaScript
  • images/ Folder
  • README.md Documentation

4 Upload Using Git Commands

The best way to upload your project is using Git commands in the terminal.

Important: You need to have Git installed on your computer for these commands to work.

# Navigate to your project folder cd cyber-forest-ctf # Initialize Git git init # Add all files to be tracked git add . # Commit your files git commit -m "Initial commit of Cyber Forest CTF" # Add the GitHub repository as origin git remote add origin https://github.com/your-username/your-repo-name.git # Push your code to GitHub git branch -M main git push -u origin main

5 Alternative: Drag and Drop

If you're not comfortable with Git commands, you can drag and drop files directly to GitHub.

1. Go to your repository on GitHub

2. Click "Add file" â†’ "Upload files"

3. Drag and drop your project folder

4. Add a commit message

5. Click "Commit changes"

Pro Tip: The drag and drop method is easier for beginners, but learning Git commands will help you in the long run.

6 Set Up GitHub Pages

Once your CTF is on GitHub, you can host it for free using GitHub Pages.

1. Go to your repository on GitHub

2. Click on "Settings"

3. Select "Pages" from the left sidebar

4. Under "Source", select "main" branch

5. Click "Save"

6. Wait a few minutes, then refresh the page

7. You'll see a link to your live website

Pro Tip: Your website will be available at https://your-username.github.io/your-repo-name/

Go to GitHub

Congratulations! Your CTF project is now on GitHub and live with GitHub Pages! Share your repository link with others so they can try your CTF challenge.
